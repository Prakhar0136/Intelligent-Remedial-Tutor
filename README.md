# 🎓 Intelligent Remedial Tutor

An AI-powered educational assistant that transforms PDF textbooks into interactive learning experiences. The system generates personalized quizzes, evaluates student responses, diagnoses misconceptions, and provides targeted remedial explanations using Retrieval-Augmented Generation (RAG).

---

## 🚀 Features

* 📚 Upload and learn from any PDF textbook
* 🧠 AI-generated multiple-choice questions
* ✅ Instant answer evaluation
* 🔍 Misconception detection and analysis
* 📖 Personalized remedial explanations
* 📊 Student performance tracking
* ⚡ Powered by Groq and LangChain
* 🗂️ ChromaDB-based Retrieval-Augmented Generation (RAG)
